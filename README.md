# Project Overview 

The Life Management System is a comprehensive life assistance tool to help track various aspects of a person’s life, such as tasks, finances, and mood, to help identify trends and improve overall quality of life. 

The current project is an MVP of a home dashboard with a lightweight task management system to test out the feasibility of the project, choose a tech stack, and start building out the infrastructure for the software. 

## 1.1 Tech Stack Selection 	
Backend and business logic: Java language (Version 26)  
Frontend GUI: Java Swing framework   
Database: no formal database will be used for simplification of the project. Text files will store data for persistent state
